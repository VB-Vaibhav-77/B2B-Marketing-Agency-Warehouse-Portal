## 🚀 Part 1: Running the Data Pipeline Locally

The entire project is structured to run with a single command. I have pre-written all the code for you, so you only need to run the orchestrator.

### Step 1.1: Open your Terminal
1. On your Windows PC, press the **Windows Key**, type **PowerShell**, and press **Enter**.
2. Make sure you are in the project folder. (You can type `cd "C:\Users\vaibh\Documents\antigravity\fervent-salk"` in PowerShell and press Enter to ensure you are in the workspace directory).

### Step 1.2: Run the Pipeline Orchestration
Copy and paste this command into PowerShell and press **Enter**:
```bash
python main.py
```
**What will happen?**
You will see a beautiful log stream in your terminal showing:
1. **Generating Raw Data**: Creating 1,000 customers, 50 products, thousands of purchase transactions, and website clickstream sessions with realistic errors (duplicates, outliers, and negative durations).
2. **Transforming Data (ETL)**: Cleaning text cases, resolving missing values, standardizing country names, removing invalid product prices, filtering corrupt session pageviews, and organizing the data into a **5-table Galaxy Schema** (Fact & Dimension tables).
3. **Database Creation**: Initializing a local relational SQLite database file (`data/apex_analytics.db`) and writing the schemas and performance indexes.
4. **Data Validation**: Running **23 automated database QA checks** (ensuring zero duplicate IDs, zero null values, referential integrity across both facts, and correct financial calculations). You will see `🎉 DATA VALIDATION SUMMARY: 100% PASSED`!

## ☁️ Part 3: Deploying to Google BigQuery (Free Cloud Sandbox)
*This is the ultimate resume booster. Showing recruiters you can load and query data in a cloud data warehouse puts you ahead of 90% of applicants!*

Google Cloud offers a **BigQuery Sandbox** that is **100% free** and requires **no credit card** to set up.

### Step 3.1: Access the BigQuery Console
1. Open your web browser and go to: https://console.cloud.google.com/bigquery
2. Log in with your standard Google (Gmail) Account.
3. If prompted, agree to the Terms of Service. Google will automatically create a sandbox environment for you.

### Step 3.2: Create a Cloud Dataset
1. In the BigQuery search pane, look for your project name (usually named something like `My First Project` or a custom ID). Click the three dots next to it and select **Create dataset**.
2. Set **Dataset ID** to: `apex_analytics`
3. Set **Data location** to: `us (multiple regions in United States)` or your local region.
4. Click **Create Dataset** at the bottom.

### Step 3.3: Upload the Clean Tables (CSVs)
We will upload the five clean tables generated by our Python ETL script:
1. Click the three vertical dots next to your new `apex_analytics` dataset and select **Create table**.
2. Under **Create table from**, select **Upload**.
3. Select the file from your computer:
   `C:\Users\vaibh\Documents\antigravity\fervent-salk\data\processed\dim_customers.csv`
4. Set the **Table name** to: `dim_customers`
5. Under **Schema**, check the box for **Auto detect (Schema and input parameters)**.
6. Click **Create table**.
7. **Repeat this exact process** for the remaining four CSV files in the `data/processed/` folder:
   * Upload `dim_products.csv` ➡️ Name it `dim_products`
   * Upload `dim_date.csv` ➡️ Name it `dim_date`
   * Upload `fact_sales.csv` ➡️ Name it `fact_sales`
   * Upload `fact_web_traffic.csv` ➡️ Name it `fact_web_traffic`

### Step 3.4: Execute SQL Queries in the Cloud (Manual Method)
1. Click the **+ Compose New Query** button in BigQuery.
2. Copy and paste the SQL code from:
   `C:\Users\vaibh\Documents\antigravity\fervent-salk\src\sql\bigquery_attribution.sql`
3. Click the **Run** button at the top.
4. **Boom!** BigQuery will query your uploaded tables in the Google Cloud and display your marketing ROI metrics in milliseconds! Take a screenshot of this workspace to show on your portfolio or LinkedIn!

### ⚡ Step 3.5: Run Queries Directly from PowerShell (Automated Method)
*Instead of copying and pasting code manually, you can run a single PowerShell command to trigger the query and pull results directly to your screen!*

1. Open **PowerShell** and navigate to your folder if not already there:
   ```bash

   ```
2. Run our custom BigQuery execution runner:

5. **Boom!** The script will run the complex attribution query in the cloud, format the data beautifully, and print the results right in your terminal! It will also save a copy of the results to `data/bigquery_attribution_results.csv`!

## 📊 Part 4: Building the Power BI Executive Dashboard

### Step 4.1: Import your Data
1. Open **Power BI Desktop** on your computer.
2. Click **Get Data** (in the Home ribbon) and select **Text/CSV**.
3. Navigate to `C:\Users\vaibh\Documents\antigravity\fervent-salk\data\processed/` and select `dim_customers.csv`. Click **Load**.
4. Repeat this for `dim_products.csv`, `dim_date.csv`, `fact_sales.csv`, and `fact_web_traffic.csv`.

### Step 4.2: Build the Galaxy Schema Model
1. Click the **Model View** icon on the far left pane of Power BI (it looks like three small blocks connected by lines).
2. Connect your tables by dragging the matching columns:
   * **Sales Connections**:
     * Drag `customer_id` from `dim_customers` and drop it onto `customer_id` in `fact_sales`.
     * Drag `product_id` from `dim_products` and drop it onto `product_id` in `fact_sales`.
     * Drag `date_key` from `dim_date` and drop it onto `order_date` in `fact_sales`.
   * **Web Clickstream Connections**:
     * Drag `customer_id` from `dim_customers` and drop it onto `customer_id` in `fact_web_traffic`.
     * Drag `date_key` from `dim_date` and drop it onto `session_date` in `fact_web_traffic`.
3. You will see a beautiful **Galaxy Schema** where your two fact tables are connected to their shared dimensions!

### Step 4.3: Add Advanced DAX Measures
1. Click the **Report View** icon on the left pane.
2. In the Home ribbon, click **Enter Data**. Name this new table `_Measures` and click **Load**.
3. Right-click the `_Measures` table in your data pane, select **New Measure**, and paste the DAX formulas provided in:
   `C:\Users\vaibh\Documents\antigravity\fervent-salk\power_bi\power_bi_playbook.md`
   *(Example: copy `Total Revenue = SUM(fact_sales[line_subtotal])` and hit enter).*
4. Create all key measures listed in the playbook (including the clickstream bounce rates and conversion metrics) to build your calculation core!

### Step 4.4: Style like a Professional (Optional)
To make your dashboard look exceptionally premium, you can import our custom dark theme:
1. In the **View** ribbon of Power BI, click the dropdown arrow in the **Themes** gallery.
2. Select **Browse for themes**.
3. Create a simple text file on your desktop, name it `ApexTheme.json`, paste the theme JSON block from the `power_bi_playbook.md` file, save it, and select it!
4. Your Power BI workspace will instantly transform into a stunning, sleek, dark-slate corporate interface. Drag in your cards, matrices, and charts, and prepare to wow hiring managers!
